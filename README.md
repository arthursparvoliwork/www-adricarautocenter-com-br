# 🚗 Adricar Auto Center - Website

🔗 Acesse o projeto: https://adricarautocenter.com.br/

## 📌 Sobre o Projeto
Este projeto consiste no desenvolvimento de um website institucional e comercial para uma empresa do setor automotivo, com foco na apresentação de veículos seminovos, captação de leads e facilitação de processos de financiamento.

O objetivo principal foi criar uma plataforma moderna, responsiva e funcional que aumente a conversão de clientes e melhore a presença digital da empresa.

---

## 🎯 Objetivos
- Exibir o estoque de veículos de forma organizada e atrativa
- Facilitar o contato com clientes interessados
- Permitir envio de dados para financiamento online
- Melhorar a credibilidade da empresa no ambiente digital

---

## ⚙️ Funcionalidades

### 🚘 Catálogo de Veículos
- Listagem de veículos disponíveis
- Informações detalhadas (modelo, ano, preço, km, câmbio)
- Interface simples e intuitiva para navegação

### 🔎 Filtros e Navegação
- Filtro por marca e características
- Organização visual para melhor experiência do usuário

### 📄 Formulário de Financiamento
- Coleta de dados pessoais e profissionais
- Simulação de crédito
- Envio estruturado de informações para análise

### 📱 Responsividade
- Adaptado para desktop e mobile
- Layout moderno e otimizado

### 📞 Contato e Conversão
- Integração com telefone/WhatsApp
- Facilidade para geração de leads

---

## 🧠 Tecnologias Utilizadas
- HTML5

- (Lovable / Builder visual para deploy e estrutura)
- Hospedagem e deploy web

---

## 🎨 UX/UI
- Design focado em conversão
- Layout limpo e direto
- Destaque para ofertas e chamadas de ação (CTA)

---

## 📈 Resultados Esperados
- Aumento de leads qualificados
- Maior visibilidade online
- Facilidade no processo de venda de veículos
